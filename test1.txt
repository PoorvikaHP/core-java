3.what is data type?and expalin types of data type.
-data type is used to know what type of data is stored inside the variable.
-there two types 
1.primitive data types-this data type stores single values in the memory directly.
-primitive datatypes are-int,char,float,double,boolean,long,short,byte.
-int datatype stores the numerical values like 4,97 etc.
-char datatype stores the characters within single quotes.ex-'a'.
-float datatype stores the numerical value with decimal points.ex-9.23f.
-double datatype stores the double the values of float with decimal points.ex-90.766.
-boolean datatype stores whether the condition is true or false.
2.non-primitive data types-this datatype stores the complex values.
-class,interface,object,array are non-primitive datatypes.

4.what is variable?explain types of variable.
-variables are the container to store the data values.
-there are 4types of variables in java
-static variable:variable declared inside the class but outside the method using static keyword.
-instance variable:declared outside the method and inside the class without static keyword.it is opposite of static variable hence called as non-satatic variable.
-local variable:declared anywhere inside the method of a class.
-parameterized variable:declared inside the parameter of the method,hence called as parameterized variable.
